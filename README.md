## 🔬 Research Interests

| Theme | Description |
|-------|-------------|
| **Glacial lake dynamics & GLOF risk** | Lake volume evolution, moraine stability, outburst triggers, downstream cascade modelling |
| **Multi-phase cryospheric flows** | Physical simulation of debris flows, glacial avalanches & GLOF cascades (r.avaflow, VolcFlow) |
| **Climate–cryosphere–ocean interactions** | Freshwater flux, stratification, thermohaline circulation, polar glacial melt impacts |

| **Climate-resilient infrastructure** | EcoDRR, mountain engineering, disaster risk reduction in fragile environments |
